## Cách chạy
* Yêu cầu MinGW-w64
* Clone repo này
```git bash
git clone https://github.com/nothin81/group03-oop.git
```
* Chạy trong git bash
```bash
mingw32-make.exe build
mingw32-make.exe run
```
* test command
```git bash
g++ -std=c++11 -Wall src/*.cpp -o build/online-cinema-ticket.exe
./build/online-cinema-ticket.exe
```
* Hoặc sử dụng `run.bat`
